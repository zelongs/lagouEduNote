#### 样式规则
##### 以内嵌式样式表为例
1. 所有css代码都必须书写在`<head>`标签内部的一对`<style>`标签内部
2. css在给某个标签设置样式前，必须使用选择器先选中标签
3. css样式的属性，属性名和属性值的键值对写法为`k:v`
4. 给每个选择器添加的样式属性都必须写在一对大括号{}之内
5. 给一个标签添加的所有需要的样式，都要在{}内部一一罗列出来
```js
p {
    color: red;
    font-size: 30px;
}
```
#### 注意事项
1. 每条属性后必须加分号，否则会报错
2. css中所有属性与属性之间对空格、换行、缩进不敏感

#### css注释语法
语法格式： `/* 中间部分为注释内容 */`
